## 技術仕様

### Tailwind設定
```javascript
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: {
          50: '#eff6ff',
          500: '#3b82f6',
          600: '#2563eb',
          700: '#1d4ed8',
        },
        gray: {
          50: '#f9fafb',
          100: '#f3f4f6',
          200: '#e5e7eb',
          // ...
        }
      },
      fontFamily: {
        sans: ['Inter', 'system-ui', 'sans-serif'],
      },
      boxShadow: {
        'card': '0 1px 3px 0 rgba(0, 0, 0, 0.1)',
        'card-hover': '0 4px 6px -1px rgba(0, 0, 0, 0.1)',
      }
    }
  }
}
```

### カードスタイル例
```tsx
const cardClassName = cn(
  // ベース
  'bg-white rounded-lg p-4',
  // シャドウ
  'shadow-card hover:shadow-card-hover',
  // トランジション
  'transition-all duration-200',
  // インタラクション
  'cursor-pointer hover:scale-[1.02]',
  // ドラッグ中
  isDragging && 'opacity-50 rotate-2'
)
```

### レスポンシブグリッド
```tsx
const boardClassName = cn(
  'grid gap-4 p-6',
  // レスポンシブ列数
  'grid-cols-1',
  'sm:grid-cols-2',
  'lg:grid-cols-4',
  // 最大幅
  'max-w-7xl mx-auto'
)
```
